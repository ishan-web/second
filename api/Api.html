Hi eSikshya,

Please use following url for SMS through API.

API URL:

https://smsml.creationsoftnepal.com/send?to=977{ mobile number }&content={messages}&token={Unique token generated by system}

Parameters:

  • {Unique token generated by system}

    It's a unique id generated by SMS system to access the SMS API.

  • { mobile number }

    Valid mobile numbers of NTC, NCELL, SmartCell

  • {messages}

    English Character message will counted 1 SMS per 160 Characters and Maximum 5 SMS can be sent in a single API call. Nepali Unicode Character message will be counted as 1 SMS per 70 Characters and Maximum 5 SMS can be sent in a single API call.
